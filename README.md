# **Notepad**

This a simple notepad created with `customtkinter` and `json` having a simple UI. Here you can add, edit and delete notes.

## **Features**
>
> **Add notes:**
> you are able to add notes with title and content
>
> **Edit notes:**
> you are able to view and edit notes
>
> **Delete notes:**
> you are able to delete notes
>
> **Save notes:**
> you are able to save notes with title and content
>
